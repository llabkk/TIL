### 기본문법 & 제어문
#### 미리 알아두기
JVM(Java Virtual Machine) : 자바 프로그램이 실행되는 환경
JRE(Java Runtime Environment) : 자바 프로그램이 실행되기 위한 환경을 제공
JDK(Java Development Kit) : 자바 애플리케이션을 개발하고 실행하기 위한 모든 도구를 제공

- System.out.printf() : 문자열을 형식화 하여 출력
